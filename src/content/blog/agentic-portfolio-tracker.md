**HODL** — an **agentic portfolio tracker** for stocks and crypto on **Base**, shipped for **START Hack**. It combines a trading UI, portfolio summaries, and **Hodlini**, an in-app assistant that helps users move from traditional equities into crypto with thematic alignment.

The architecture uses a **browser event bus** so trades update positions and the dashboard stays in sync without fragile manual refreshes — illustrated below — and we took **Demo Winner** recognition on the START Hack Campus stage.
