Built for **START Hack 2026** (`thack-2026.vercel.app`), this project is an **AI data analytics board** — a dark, conversational **ZR Analytics**–style experience: particle “brain” visualizations, quick actions (analyze test data, compare batches, explain results), and routes to expert agents.

The interface greets with context-aware copy and keeps the workflow centered on **lab data exploration** — matching what we pitched on stage and in demo rounds.
